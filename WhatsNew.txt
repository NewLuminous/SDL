
This is a list of major changes in 1010!'s version history.

---------------------------------------------------------------------------
1.1.0:
---------------------------------------------------------------------------

* The preview of the piece before placing it onto the board now shows its border instead.
This indicates whether the chosen piece overlaps other placed pieces.
* Increase the pick range


---------------------------------------------------------------------------
1.2.0:
---------------------------------------------------------------------------

* The chosen piece sticks to the cursor instead of disappearing.


---------------------------------------------------------------------------
1.3.0:
---------------------------------------------------------------------------

* Add audio and option to turn if off.
* Add pause menu.


---------------------------------------------------------------------------
1.4.0:
---------------------------------------------------------------------------

* The current score and game status are saved before the game being closed.
The player can continue the game by reopening it.
* Fix bug where new pieces are not randomly generated.


---------------------------------------------------------------------------
1.5.0:
---------------------------------------------------------------------------

* Add animations when completing lines and generating new pieces.
* Highlight button on menu which is being pointed at.
* The player now gets more score when completing multiple lines at once.
